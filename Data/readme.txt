ScienceQA contains examples of Context, Question and Answer triples from Scholarly articles. It has ~100K such triples. We devide the whole dataset into train, Development/Validation and Test set. So there are three files:
--train.csv #(75429, 2)
--val.csv #(9428, 2)
--test.csv #(9429, 2)
This dataset is for evaluating end-to-end neural Question Answering and Question Generation modeuls specially in the scientific domain.
